# Personal Site

This is a minimal, accessible, and unstyled personal website built with Vue 3. It features a simple homepage and a custom background component.

## Tech Stack

- [Vue 3](https://vuejs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [gsap](https://greensock.com/gsap/) (for animation)
- [three.js](https://threejs.org/) (for 3D rendering)

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run the development server:
   ```sh
   npm run dev
   ```
3. Open your browser to the local server URL (http://localhost:5173).

## Project Structure

- `src/App.vue` — Root app shell, renders the homepage and background
- `src/views/HomePage.vue` — Main homepage
- `src/components/TheBackground.vue` — Custom background component
- `public/` — Static assets and images

## Accessibility & Responsiveness

- All interactive elements are keyboard accessible
- Layout adapts to all screen sizes
